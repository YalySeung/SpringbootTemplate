# 📦 Spring Boot Template Project

> Spring Boot 3.1.5 + JDK 17 + Gradle 기반 템플릿 프로젝트  
> 기본 CRUD, Swagger API 문서, QueryDSL 설정 포함

---

## 🚀 기술 스택

| 구분      | 내용                                       |
|-----------|--------------------------------------------|
| Language  | Java 17                                    |
| Framework | Spring Boot 3.1.5                          |
| Build     | Gradle (Groovy DSL)                        |
| ORM       | Spring Data JPA + Hibernate                |
| DB        | H2 (테스트용) / MariaDB / Oracle 설정 가능 |
| Docs      | Swagger (springdoc-openapi 2.2.0)          |
| Util      | Lombok, QueryDSL 5.0.0                     |
